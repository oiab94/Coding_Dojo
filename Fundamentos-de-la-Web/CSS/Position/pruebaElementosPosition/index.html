Propiedades de position

position: static

Los elementos en HTML son posicionados por defecto en static. Las propiedades posicionadas con static no son afectados por las propiedades top, bottom, left, right.

Este es un div con posicion static.

position: relative

Una posicion relative es posicionado relativamente con referencia a su posicion normal. Utilizando las propiedades left, right, bottom, top ajustara la posición del elemento con respecto a su posición normal.

Este es un div con posicion relative

position: fixed

Una posición fixed se encuentra posicionada con respecto a la vista, lo que indica que siempre se encuetra ubicada en la misma posición sin importar si la pagina es movida con el scroll. Las propiedades top, bottom, right, left son utilizadas para posicionar el elemento.

Este es un div con posicion fixed

position: absolute

Una posición absolute es posicionada en una posición más cercana a su ancestro (similar a fixed que se posiciona con respecto a la vista). El contenedor padre debe estar configurado como relative por que, el la posición absolute buscara el elemento ancestro más cercano con posición relative normalmente, suele ser el elemento body.

Este elemento se encuentra con posición relative.

Este es un div con posicion absolute
